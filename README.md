<h1 align="center"> WaveCast </h1>

## Bem-vindo ao WaveCast!

O WaveCast é um site com a temática de **podcasts para desenvolvedores**. Este projeto foi desenvolvido como parte de um curso oferecido pela **Alura**. 

## Sobre o Projeto

O WaveCast é um projeto estático é expirado no Spotify e a inovação no mundo dos podcasts voltados para desenvolvedores. Com um design moderno, este site apresenta uma experiência imersiva para amantes de tecnologia e programação.
